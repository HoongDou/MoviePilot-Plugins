
# UBits 自动签到

自动签到 [UBits.club](https://ubits.club)，基于 MoviePilot V2 插件框架。

## 配置说明

| 配置项 | 说明 |
|--------|------|
| 启用插件 | 开启后定时任务生效 |
| 发送通知 | 签到后推送结果到已配置的通知渠道 |
| 立即运行一次 | 保存后立即执行一次签到，执行后自动关闭 |
| 使用代理 | 是否通过系统代理访问 UBits |
| 执行周期 | 5位 cron 表达式，推荐 `0 8 * * *`（每天早8点） |
| Cookie | 登录 UBits 后从浏览器复制的 Cookie |
| User-Agent | 留空使用系统默认 UA |

## 获取 Cookie

1. 登录 [UBits.club](https://ubits.club)
2. 按 `F12` 打开开发者工具
3. 切换到 **Network** 标签
4. 刷新页面，点击任意请求
5. 在请求头中找到 `cookie:` 字段，复制完整内容

## 注意事项

- Cookie 有效期有限，失效后需重新获取并填写
- 签到成功不等同于站点认定为活跃，请结合 UBits 公告自行判断
- 详情页可查看最近 30 条签到记录
